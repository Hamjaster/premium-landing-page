# Project Design Context

## Users
- **Audience**: Visitors evaluating a premium Nike shoe brand experience.
- **Context**: They expect a polished, cinematic first impression that signals quality and performance immediately.
- **Job to be done**: Understand the shoe collection fast and feel compelled to explore or purchase.
- **Emotional goals**: Confidence, excitement, trust, and exclusivity.

## Brand Personality
- **Voice**: Bold, energetic, and authoritative without feeling distant.
- **Tone**: Athletic, cinematic, modern.
- **3-word personality**: Bold, Cinematic, Athletic.
- **Emotional goals**: Performance, craftsmanship, and premium credibility.

## Aesthetic Direction
- **Visual tone**: Dark, cinematic, atmospheric, and premium — deep red/maroon base with vibrant red and orange accents.
- **Primary colors**: Deep Maroon (#1A0A0A), Red (#E63946), Orange (#FF6B35), Amber (#FFB800).
- **Typography**: Satoshi family with strong hierarchy and restrained contrast.
- **Theme**: Dark mode only.
- **References**: Nike.com, ElevenLabs, RunwayML.
- **Anti-references**: Generic SaaS layouts, bright palettes, cluttered interfaces, playful consumer styling.

## Design Principles
1. **Cinematic first**: Treat each section like a scene with a clear focal point.
2. **Atmosphere over decoration**: Use gradients and light to build depth, not surface noise.
3. **Typography leads**: Headlines carry the message and visual weight.
4. **Glass and contrast**: Use subtle glass layers and borders to separate surfaces.
5. **Motion with purpose**: Animate only when it clarifies or elevates the experience.
6. **Measured rhythm**: Keep spacing, alignment, and transitions disciplined.

## Implementation Constraints
- **Framework**: React + TypeScript + Vite.
- **Styling**: Tailwind CSS only for future UI work in this project.
- **Animations**: Framer Motion and GSAP are available if needed, but should stay restrained.
- **Font**: Satoshi via Fontshare CDN.
- **Theme**: Dark mode only.

## Color Tokens
- **Backgrounds**: bg-primary (#1A0A0A), bg-secondary (#2A0F0F), bg-tertiary (#3D1515).
- **Red accents**: red-primary (#E63946), red-light (#FF5C6A), red-dark (#B82D38).
- **Orange accents**: orange-primary (#FF6B35), orange-light (#FF8F5E), orange-dark (#CC5529).
- **Amber accents**: amber-primary (#FFB800), amber-light (#FFD166), amber-dark (#CC9300).
- **Text**: text-primary (#FFFFFF), text-secondary (#B0A0A0), text-tertiary (#7A6A6A), text-accent (#FF8F8F).
- **Borders**: border-subtle, border-medium, border-strong.
- **Glass**: glass-bg, glass-border.
