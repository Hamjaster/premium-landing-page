# Project Design Context

## Users
- **Audience**: Visitors evaluating a premium luxury brand experience.
- **Context**: They expect a polished, cinematic first impression that signals quality immediately.
- **Job to be done**: Understand the brand fast and feel compelled to engage.
- **Emotional goals**: Confidence, awe, trust, and exclusivity.

## Brand Personality
- **Voice**: Confident, refined, and authoritative without feeling distant.
- **Tone**: Sophisticated, cinematic, modern.
- **3-word personality**: Luxurious, Cinematic, Refined.
- **Emotional goals**: Exclusivity, craftsmanship, and premium credibility.

## Aesthetic Direction
- **Visual tone**: Dark, cinematic, atmospheric, and premium.
- **Primary colors**: Black (#000000), Gold (#63522B), Teal (#08261F).
- **Typography**: Satoshi family with strong hierarchy and restrained contrast.
- **Theme**: Dark mode only.
- **References**: RunwayML, ElevenLabs, SpaceX.
- **Anti-references**: Generic SaaS layouts, bright palettes, cluttered interfaces, playful consumer styling.

## Design Principles
1. **Cinematic first**: Treat each section like a scene with a clear focal point.
2. **Atmosphere over decoration**: Use gradients and light to build depth, not surface noise.
3. **Typography leads**: Headlines carry the message and visual weight.
4. **Glass and contrast**: Use subtle glass layers and borders to separate surfaces.
5. **Motion with purpose**: Animate only when it clarifies or elevates the experience.
6. **Measured rhythm**: Keep spacing, alignment, and transitions disciplined.

## Implementation Constraints
- **Framework**: React + TypeScript + Vite.
- **Styling**: Tailwind CSS only for future UI work in this project.
- **Animations**: Framer Motion and GSAP are available if needed, but should stay restrained.
- **Font**: Satoshi via Fontshare CDN.
- **Theme**: Dark mode only.

## Color Tokens
- **Backgrounds**: bg-primary (#000000), bg-secondary (#0a0a0a), bg-tertiary (#111111).
- **Gold accents**: gold-primary (#63522B), gold-light (#8B7340), gold-dark (#4A3D1F).
- **Teal accents**: teal-primary (#08261F), teal-light (#0D3D31), teal-dark (#051A15).
- **Text**: text-primary (#FFFFFF), text-secondary (#A0A0A0), text-tertiary (#666666), text-gold (#C9A96E).
- **Borders**: border-subtle, border-medium, border-strong.
- **Glass**: glass-bg, glass-border.
